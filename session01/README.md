# Session01: 环境搭建与Hello World

## 学习目标
- 掌握Python的安装与配置
- 了解IDE的选择与设置（VS Code/PyCharm）
- 编写第一个Python程序

- Python环境搭建
- IDE配置与使用
- Python程序基本语法
- 变量的基本概念
- 输入输出函数的使用
- 注释的编写规范

交互式计算器 - 一个能够进行基本四则运算的命令行计算器程序

## 前置知识
- 基本的计算机操作能力
- 了解什么是编程语言（可选）

- `tutorial.md` - 详细教程文档

## 快速开始

```bash
# 进入session01目录
cd session01

# 运行演示代码
uv run python demo.py

# 运行演示项目
cd project
uv run python main.py
```
