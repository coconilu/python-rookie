关于本站

欢迎来到我的个人博客！

这是一个使用 Flask 框架搭建的个人博客系统。在这里，我会分享我的学习笔记、技术心得和生活感悟。

关于我

我是一名热爱编程的开发者，正在学习 Python 和 Web 开发。通过这个博客项目，我希望能够：

  • 记录自己的学习历程
  • 分享技术经验和心得
  • 与其他开发者交流学习
  • 不断提升自己的技术水平

技术栈

本博客系统使用了以下技术：

  • 后端框架：Flask (Python)
  • 模板引擎：Jinja2
  • 数据存储：JSON 文件（适合小型博客）
  • 前端：HTML5 + CSS3 + JavaScript
  • 部署：可部署在任何支持 Python 的服务器上

功能特性

  • 文章的创建、编辑、删除
  • 标签系统，方便文章分类
  • 全文搜索功能
  • 文章浏览计数
  • 响应式设计，支持移动设备
  • 简洁美观的界面
