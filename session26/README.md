# Session26: 项目需求分析与设计

- 掌握软件项目需求分析的基本方法
- 学会编写规范的需求文档
- 理解系统设计的基本原则
- 掌握UML建模基础
- 学会制定项目开发计划

- 需求工程基础

- 用例图设计
- 系统架构设计
- 数据库设计
- 项目计划制定

## 前置知识

- 面向对象编程

- 软件工程概念
